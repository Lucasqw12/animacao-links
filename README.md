# ✨ Animação de Links

Projeto desenvolvido com **HTML e CSS** com foco em **animações em links**, criando efeitos visuais modernos e interativos para navegação.
O objetivo é demonstrar como transformar links simples em elementos mais dinâmicos utilizando apenas CSS.



## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Transitions
* Hover Effects
* Flexbox


## 🎯 Objetivo do Projeto

Este projeto foi criado para praticar:

* Animações com CSS
* Transições suaves
* Estilização de links
* Boas práticas de CSS
